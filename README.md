# PixelNest - Premium Web Development Website

A modern, high-conversion website for PixelNest - a web development and digital solutions company.

## Features

- **Modern Design**: Ultra-modern, premium tech startup aesthetic
- **Dark/Light Theme**: Toggle between themes with smooth transitions
- **Fully Responsive**: Mobile-first design that works on all devices
- **Smooth Animations**: Micro-interactions and scroll animations
- **SEO Optimized**: Structured with SEO best practices
- **Fast Performance**: Optimized for speed and Core Web Vitals
- **Glassmorphism**: Modern glass effects and gradients

## Pages

1. **Home** - Hero section, services overview, testimonials, founder section
2. **About** - Mission, vision, company story, leadership team
3. **Services** - Detailed service offerings with benefits and results
4. **Portfolio** - Project showcase with before/after results
5. **Testimonials** - Client success stories and reviews
6. **Contact** - Contact form and business information

## Tech Stack

- React 18
- React Router DOM
- CSS3 with CSS Variables
- Modern JavaScript (ES6+)

## Installation

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm start
```

3. Build for production:

```bash
npm run build
```

## Customization

### Colors

Edit CSS variables in `src/styles/index.css`:

- `--primary-blue`: Main brand color
- `--purple`: Secondary accent
- `--neon-accent`: CTA highlights

### Content

Update content directly in page components:

- Company info in About.js
- Services in Services.js
- Projects in Portfolio.js
- Testimonials in Testimonials.js
